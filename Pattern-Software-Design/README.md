# Pattern Software Design Post Training Qualification Answer

Link Repository:
https://github.com/StyNW7/PatternSoftwareDesign-Qualification

Material:
- Full Stack Development using ASP.NET C#
- Domain Driven Design (DDD) Architecture
- ASP.NET Database
- Crystal Report

Rubric Scoring:
1. Minimum 2 master tables and 1 transaction (header detail)
2. Minimum 2 roles (admin & guest)
3. Login page
4. Register page
5. Transactions use carts (session/cookie/database)
6. CRUD carts
7. CRUD for each master table
8. Use web services
9. Use DDD concepts
10. Have viewstate (Authentication & Authorization)
11. Reporting (calculations)
